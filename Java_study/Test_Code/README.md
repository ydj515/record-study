# Test Code

## JUnit5
- 자바용 단위 테스트 도구

assertArrayEquals(a, b); 
 - 배열 A와 B가 일치함을 확인한다.

assertEquals(a, b);
 - 객체 A와 B가 일치함을 확인한다.

assertEquals(a, b, c);
 - 객체 A와 B가 일치함을 확인한다.
 - a: 예상값, b:결과값, c: 오차범위

assertSame(a, b); 
 - 객체 A와 B가 같은 객임을 확인한다.

assertTrue(a);
 - 조건 A가 참인가를 확인한다.

assertNotNull(a); 
 - 갹채 A가 null이 아님을 확인
